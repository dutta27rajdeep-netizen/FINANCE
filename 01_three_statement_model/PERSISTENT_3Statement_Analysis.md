```
INCOME STATEMENT                    BALANCE SHEET

  │                                   ▲
  ├─ Net Income ₹694 Cr              │
  │    ↓                              │
  └──→ Retained Earnings          Assets = Liabilities + Equity
       increase on BS                 │
                                    Equity grows via:
                                    + Net Income
                                    - Dividends Paid
                                    = New Retained Earnings
```

### Linkage Verification Checklist

| Linkage | Verification | Status |
|---|---|---|
| **Net Income → RE Growth** | RE increased by ₹694 Cr (net income) less dividends | ✅ VALID |
| **Depreciation Bridge** | DA adds back ₹200 Cr non-cash expense in CF calculation | ✅ VALID |
| **Free Cash Flow** | FCF = OCF (₹1,044 Cr) - CapEx (₹250 Cr) = ₹794 Cr | ✅ VALID |
| **Cash Consistency** | Cash on BS = Prior cash + FCF - Financing outflows | ✅ VALID |
| **Balance Sheet Equation** | Assets (₹3,350 Cr) = Liabilities (₹725 Cr) + Equity (₹2,625 Cr) | ✅ VALID |
